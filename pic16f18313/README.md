# PIC16F18313用
プリント基板を入手された方はマイコンの向きに注意して写真のように組み立ててください。5ピンのピンソケットをL字に曲げてハンダ付けします。プログラムの書き込みは、PICkitなどを使ってICSP端子から行ってください。その際、1ピン～5ピンだけを使います。6ピンは使わないのでスルーホールはありません。
